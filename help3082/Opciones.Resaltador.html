Puede establecer varios esquemas de resaltado de sintaxis en esta pagina. Cada elemento de codigo tiene 3 propiedades: primer plano, fondo y estilo.
Hay 3 elementos de estilo disponibles:

  • B - negrita
  • I - cursiva
  • U - subrayado

El esquema de resaltado seleccionado tambien se usa para guardar el codigo fuente en el formato RTF o HTML.

Elementos del codigo:

Comentarios: el texto despues de doble barra // o entre llaves {}.

Etiquetas: los identificadores que comienzan con un @.

Variables: los identificadores que tienen un signo de dolar $ al principio (variables globales) o que terminan con un @ (variables locales). Tambien el tipo de ADMA (comienza con un &) y las matrices se resaltan igual que las variables.

Palabras clave: las palabras especiales almacenadas en el archivo keywords.ini (una para todos los juegos) y keywords.txt (archivo separado para cada juego).

Numeros: el numero entero y el numero flotante

Cadenas: el texto entre "..." y "...".

Modelos: los identificadores comienzan con un #.

Clases: la parte del comando de clase: el identificador antes del punto.

Comandos: la parte de los comandos de clase: identificador despues del punto.

Directivas: el comentario que contiene un $ inmediatamente despues de la apertura {.
