Un cuadro de texto es un cuadrado negro en la esquina superior izquierda con letras blancas
Se hace con este opcode:

03E5: show_text_box 'HELP101'

Este es el cuadro de texto 'normal'. Desaparecera despues de algún tiempo.

0512: show_permanent_text_box 'HOSP_1'

Este es un cuadro de texto permanente, por lo que se mostrara hasta que se ejecute un codigo de operacion que lo elimine. Ese es el siguiente codigo de operacion:

03E6: remove_text_box

Por ejemplo: 

0512: show_permanent_text_box 'HOSP_1'
wait 1000
03E6: remove_text_box

Con este codigo de operacion, visualizara un cuadro de texto y lo eliminara despues de 1 segundo. 

0513: show_text_box_1number 'SLOT_02' number 2

Este es un codigo de operacion para un cuadro de texto con un número en el texto, en este caso el valor 2.
Hay tres codigos de operacion que verifican si el cuadro de texto existe: 

0A2A: text_box 'SGPUNT' displayed
08FE: text_box_displayed
0989: set_text_boxes_width 500

Con este codigo de operacion, puede editar el ancho del cuadro de texto. Para ver la diferencia, puede utilizar el siguiente codigo: 

0512: show_permanent_text_box 'HOSP_1'
wait 1000
0989:  set_text_boxes_width 500
