Un modo de edicion es un conjunto de opciones que determinan el comportamiento de Sanny Builder cuando se trabaja con scripts de un juego en particular. El hecho es que no hay ningun juego en la serie de GTA con el formato de script igual a los demas. Por lo tanto, cada juego requiere una configuracion especial al editar scripts. Esa es la razon por la que los modos de edicion se presentan en Sanny Builder.

Un modo de edicion incluye (pero no se limita a) las siguientes opciones:

  • Sanny Builder desarma un archivo de script de una manera especifica
  • Sanny Builder puede compilar un script para el juego dado
  • Puedes ejecutar el juego y leer/escribir las coordenadas del jugador

Por defecto, hay 7 modos diferentes:

  • GTA III
  • GTA VC
  • GTA SA (para codigos de operacion personalizados)
  • GTA LCS
  • GTA VCS
  • SA Mobile (para las versiones de Android e iOS)
  • GTA SA SCR (para nombres R* originales)

Puede agregar un nuevo modo editando manualmente el archivo games.xml ubicado en el directorio <SB>\data\.

Para cambiar el modo de edicion, haga clic en en la esquina inferior derecha de la ventana principal de Sanny Builder. Aparecera una lista de los modos disponibles. Elija uno y haga clic en el. Sanny Builder volvera a cargar todos los archivos necesarios y puede comenzar a trabajar.

Antes de desmontar o compilar un script, asegurese de que el modo de edicion correcto este activo. Incluso si el script se compila sin errores, el juego se bloqueara cuando intente leer un script en un formato diferente.

Un archivo de script compilado puede tener una informacion para que juego esta hecho. Por lo tanto, si descompila una secuencia de comandos de ese tipo en otro modo de edicion que este activo, Sanny Builder le propondra que cambie la modalidad a la correcta. Si se rechaza este aviso, puede producirse un bloqueo del desensamblador, ya que el formato del script es inesperado.
