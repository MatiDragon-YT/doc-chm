Una matriz representa una colección indexada de elementos del mismo tipo (llamado tipo base). Puede trabajar con cualquier elemento directamente a través de su índice. Una numeración de índice comienza con un cero. Las matrices son compatibles con San Andreas, LCS y VCS.

Sintaxis general

San Andreas
<nombre>(<indice>,<tamaño><tipo>) 
$indice = 0
$arreglo($indice,10i) = 1
  • <nombre> - variable local o global para el nombre del arreglo
  • <indice> - cualquier variable que contenga un índice del elemento para leer o escribir
  • <tamaño> - cualquier valor mayor que 0
  • <tipo> - uno de los caracteres i f s v:
Letra Tipo de artículo Tamaño del artículo (bytes)
i integer 4
f float 4
s string 8
v string 16
Liberty City Stories, Vice City Stories
<nombre>(<indice>,<tamaño>)
En LCS, VCS, los elementos de la matriz tienen solo 4 bytes de longitud. Por lo tanto, no es necesaria una declaración de tipo.
$indice = 0
$arreglo($indice,10) = 1

Declaración de arreglo

Los arreglos se pueden declarar utilizando el constructor VAR..END
Sintaxis
<nombre>: array <tamaño> of <tipo> 
var
    $FloatArray: array 10 of Float
end
Acceder a los elementos del arreglo después de la declaración

Después de declarar un arreglo, puede acceder a sus elementos utilizando corchetes:

var
    $FloatArray: array 10 of Float
end
$index = 1
$FloatArray[$index] += 100.0
Usando índices constantes

Puede usar números enteros positivos y cero para acceder a un elemento del arreglo en particular:

var
    $FloatArray: array 10 of Float
end
$FloatArray[1] += 100.0

Se puede acceder a los elementos de un arreglo que contiene cadenas literarias con variables de cadena:

// inicializando los primeros tres elementos del arreglo $ strings
s$strings[0] = 'str1'
s$strings[1] = 'str2'
s$strings[2] = 'str3'

Un arreglo se puede declarar como una colección de instancias de clase:

var
  $players: array 2 of Player
end
$players[0].Build
De forma predeterminada, en el modo de edición de GTA SA, el desensamblador imprime elementos de matriz con los números como índices. La misma función está disponible en LCS y VCS, pero está desactivada de forma predeterminada. Puede activar y desactivar esta función mediante la opción de depuración CONSTANT_INDEXES
