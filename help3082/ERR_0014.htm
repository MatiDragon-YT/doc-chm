Sanny Builder Error Messages

 

Incorrect expression

This error message is given, if you use a math expression without specifying an opcode, and the operands types are incompatible. The operands incompatibility is possible when you:

a) declared a variable of a type using VAR..END, but in fact used this variable with another type;
b) both of the variables weren’t declared through the VAR..END;
c) one of the expression operands is impossible. For example, you used a number as the left operand of an assignment statement. (5=6).

Possible solutions: check correctness of this math expression. Choose a solution depending on the kind of error:

a) re-declare the variable with needed type;
b) declare both variables, using VAR..END;
c) fix an expression using possible values.
