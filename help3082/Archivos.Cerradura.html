Liberación

Por lo general los archivos abiertos, son cerrados automaticamente cuando se cierra el juego
Sí quieres cerrarlos antes para mantener la memoria RAM más limpia, podrias usar este opcode:

0A9B: closefile $hFILE
Parametros
1 - Variable contenedora del archivo.

Destrucción

Es posible tambien destruir los archivos, con este opcode 
0B00: delete_file "CLEO\example.txt"  // IF and SET
Tambien puede ser usado como condición:
Sí el archivo se elimina correctamente, se devuelve un valor cero.
En caso de error, se devuelve un valor distinto de cero.
