LPSTR WINAPI CLEO_ReadStringOpcodeParam(CScriptThread* thread, LPSTR buf, int size);

Descripcion:
La funcion lee del script 1 un operando de tipo String en el bufer y devuelve un puntero a este bufer.

Parametro Descripcion
thread Puntero de estructura de flujo
buf Puntero a la cadena en la que se escribira el resultado. Al pasar NULL, la cadena del script se leera en el bufer interno y se devolvera un puntero a este bufer.
size Peso de la cadena en bytes.

Ejemplo de uso:

char * str = CLEO_ReadStringOpcodeParam(thread, NULL, 0);
