Sanny Builder admite atajos de teclado estándar para las operaciones de archivos y texto (Ctrl + N, Ctrl + Z, etc.). Además, hay teclas rápidas específicas para usar en el trabajo con scripts. Algunos de ellos son personalizables en las opciones

Menú
F5 abrir archivo como script y descompilar
F6 compilar texto en un archivo de script
F7 compilar texto en un archivo de script y copiarlo en el directorio de scripts para el juego actual
F8 ejecutar el juego actual
F10 abre la ventana de opciones
F12 abra el archivo de ayuda (este)

Texto
TAB , Shift+TAB mover el texto seleccionado a la izquierda / derecha (tabmove el texto seleccionado a la izquierda / derecha (pestaña)
Ctrl+Alt+A / D mueve el texto seleccionado hacia la izquierda / derecha (con un símbolo)
Ctrl+Alt+B / N / M diferentes modos de selección de texto: Normal, Columnar (igual que la selección con Alt presionado), Líneas
Ctrl+Shift+U / L hacer palabra mayúscula / minúscula
Ctrl+Num+ aumentar el tamaño del texto (acercar)
Ctrl+Num- disminuir el tamaño del texto (alejar)
Ctrl+T borrar palabra
Ctrl+Y borrar linea
Ctrl+Shift+Y línea clara
Ctrl+Q línea de comentario / comentario

Cursor
Ctrl+Shift+B saltar entre paréntesis: {}, (), <>, []
Ctrl+Shift+Num8 / Num2 desplazarse página en línea arriba / abajo
Ctrl+Scroll Up / Down página arriba / abajo
Ctrl+Shift+0..9 alternar un marcador
Ctrl+0..9 saltar al marcador
Ctrl+G ir a linea
Ctrl+Num4 / Num6 busca la palabra debajo del cursor debajo / arriba en el código
Ctrl+Num2 saltar a la etiqueta debajo del cursor
Ctrl+Num8 devuelve el cursor a la posición anterior (antes de InstaSearch)
Ctrl+Enter abrir el archivo cuyo nombre está debajo del cursor (ver también las prioridades relativas de la ruta)

Código
Ctrl+[space] mostrar lista de clases / clase miembros / modelos / etiquetas / variables / misiones
Ctrl+Shift+[space] mostrar la lista de parámetros para el comando de clase actual
Ctrl+Shift+C insertar las coordenadas del jugador
Ctrl+Shift+E insertar el angulo del jugador
F1 buscar código de operación con la palabra debajo del cursor (ignorando '_' y '.')
Ctrl+F1 buscar código de operación con la palabra debajo del cursor (sin ignorar '_' y '.')
F2 usar macro
Ctrl+J mostrar la lista de macros
Ctrl+M iniciar / detener la grabación de una macro (secuencia de pulsaciones de teclas)
Ctrl+P reproducir macro / pausa mientras procesa
Ctrl+H convertir un número debajo del cursor de decimal a hexadecimal y viceversa
Ctrl+Alt+H convertir un nombre de modelo bajo el cursor a su ID numérico y viceversa

Misc
Ctrl+~ abrir consola
Alt+S (cuando la lista de modelos está activa) reordenar la lista
