Un operador es un símbolo que realiza manipulaciones matemáticas o lógicas.

Operadores Aritméticos:

Sanny Builder soporta los siguientes operadores aritméticos:

Descripción: Simbolo Forma
Asignar = x = y
Adición + x += y
Substracción - x -= y
Multiplicación * x *= y
Divición / x /= y
Modulo % x %= y
int $x = 10
int $y = 5
print $x * $y
