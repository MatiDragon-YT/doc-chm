Los comandos adicionales son para acelerar la produccion de siertas funciones, como generar numeros random, entre otras cosas.

INC

Incrementa el primer parametro por el segundo. El primer parametro es una variable. 

Inc($IntVar1, $Value)
=
$IntVar1 += $Value

El segundo parametro es igual a 1 si no se especifica.

Inc(1@)
=
1@ += 1

Desde la version 3.06, es posible usar un operador ++ similar a C++ que se puede aplicar a las variables e incrementa el valor que tienen en uno.

$var1++
=
$var1 += 1

DEC

Disminuye el primer parametro por el segundo. El primer parametro es una variable. 
Dec($IntVar2, $Value)
=
$IntVar2 -= $Value

El segundo parametro es igual a 1 si no se especifica.

Dec(1@)
=
1@ -= 1

Desde la version 3.06 es posible usar un operador similar a C++, que se puede aplicar a las variables y disminuye el valor que tienen en uno.

$var--
=
$var -= 1

MUL

Multiplica el primer parametro por el segundo. El primer parametro es una variable. 

Mul($IntVar3, $Value)
=
$IntVar3 = $IntVar3 * $Value

El segundo parametro es igual a 2 si no se especifica.

Mul(1@)
=
1@ = 1@ * 2

DIV

Divide el primer parametro por el segundo. El primer parametro es una variable. 

Div($IntVar4, $Value)
=
$IntVar4 = $IntVar4 / $Value

El segundo parametro es igual a 2 si no se especifica.

Div(1@)
=
1@ = 1@ / 2

(interno) ALLOC*


Esta funcion establece la direccion de memoria para la variable global. Se debe usar solo para aquellos con nombres personalizados (es decir, $text). Para otros, la direccion de memoria se establece por su nombre: la variable $40 siempre se dirige a la direccion de memoria #40.

Alloc($MyVar5, 40) // la variable $MyVar5 pasa a compilarse como $40
(!) Ver tambien 'help\examples\alloc.txt'

SQR


Calcula el cuadrado de una variable pasada como parametro*. Se puede usar solo como procedimiento.

sqr($var6) 
=
$var6 *= $var6

* tipo de variable tiene que ser definido.

RANDOM


Esta funcion genera un numero aleatorio dentro de un rango especificado por los parametros.

$rnd = random(1, $high)

Esta funcion se puede usar para enteros y flotantes. El codigo de operacion se elige de acuerdo con el tipo de variable de resultado ($rnd en este caso)
