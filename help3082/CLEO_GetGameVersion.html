int WINAPI CLEO_GetGameVersion();

Descripcion:
La funcion devuelve el identificador de la version del juego:

  • GV_US10 - US version 1.0
  • GV_US11 - Version de EE. UU. 1.01, esta version no es compatible con CLEO
  • GV_EU10 - Version europea 1.0
  • GV_EU11 - Version europea 1.01
  • GV_UNK  - Cualquier otra

Ejemplo de uso:

if (CLEO_GetGameVersion() != GV_US10) {
  MessageBox(HWND_DESKTOP, "Esta version del juego no es compatible.", NULL, MB_ICONERROR);
  return FALSE;
}
