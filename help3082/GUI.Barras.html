Puedes encontrar barras como medidores de estados como en las misiones "Drive-By" y "supply-lines"
Estas se muestran con valores que van del 0 al 100, para mostrar de sierta forma un porcentajes.

03C4: $health2 = create_counter_type 1 with_GXT 'SWE4_08' // CAR HEALTH
Parametros
  • $health2 - variable para almacenar y manipular (unicamente del tipo global)
  • 1 - Forma para mostrar el contador, con 0 en numeros, y con 1 en barra de porcentaje
  • 'SWE4_08' - ID del texto GXT
Estos se crean unicamente con variables globales, por que estos no desaparecen hasta que los destruyas con el opcodes 0151 
03C4: $counter = create_type 1 with_GXT 'NULL'
$counter = 100
repeat
    wait 50 ms
    $counter -= 1
until  $counter < 0
0151: remove_status_text $health2
