void WINAPI CLEO_SetThreadCondResult(CScriptThread* thread, BOOL result);

Descripcion:
La funcion establece el resultado de una operacion logica para una secuencia condicional boleana (TRUE o FALSE).

Parametro Descripcion
thread Puntero de estructura de flujo
result TRUE o FALSE.

Ejemplo de uso:

CLEO_SetThreadCondResult(thread, a != 0);
