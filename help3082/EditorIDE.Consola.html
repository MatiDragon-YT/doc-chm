La consola tiene la intencion de cambiar las opciones avanzadas de Sanny Builder. En su mayoria se relacionan con las caracteristicas del descompilador y puede ver los cambios al descompilar un archivo. La ventana de la consola se invoca presionando Ctrl + ~. Puede escribir los comandos y los parametros en la linea de edicion. Despues de escribir un comando, presione Enter para procesar. La consola mostrara un resultado.

Todos los comandos no distinguen entre mayusculas y minusculas.

Los comandos disponibles son:

help
muestra la informacion de ayuda sobre la consola

clear
borra la consola

list
muestra en la consola la lista de todos los comandos

El parametro (opcional):

lista

options
despues de las opciones de la lista de entrada, la consola mostrara todas las opciones posibles para alternar.

toggle
el comando principal para habilitar o deshabilitar las opciones del programa

El parametro:

alternar

<nombre de opcion> [on/off]

La lista de opciones puede ser diferente en otra version. Para obtener todas las opciones posibles, escriba opciones de lista.

La linea de comando puede finalizar con la palabra ON (la opcion se habilitara) o la palabra OFF (la opcion se desactivara), o nada (la opcion se cambiara de ENCENDIDO a APAGADO o de APAGADO a ENCENDIDO)..

Nota: En primer lugar, el modo de depuracion deberia estar habilitado con el comando alternar debug_mode

toggle CODE_OFFSETS

el decompilador imprimirá las compensaciones de cada código de operación

toggle VAR_COUNTER
después de compilar, la consola contendrá la lista de las variables globales utilizadas una vez en el código. Es muy útil buscar las variables inútiles.
toggle IGNORE_UNKNOWN
hace que el decompilador funcione de manera avanzada, ignorando códigos de operación faltantes, parámetros incorrectos, etc. Puede ayudarte. Permite descompilar casi todos los archivos para descifrar los archivos protegidos (o los archivos que se compilaron incorrectamente).
toggle CONSTANT_INDEXES
permite descompilar las matrices con el índice de número constante, si es posible. Está disponible para San Andreas (habilitado por defecto), LCS, VCS. Por ejemplo, antes despues $10 $10[0] $11 $10[1] $12 $10[2]
toggle SKIP_SCM_HEADER
Hace que el descompilador omita el encabezado scm durante el procesamiento. Permite descompilar las scms externas del script.img por separado. También hace que el compilador no compile el encabezado scm.

El uso del parametro \debug es una forma alternativa de cambiar las opciones avanzadas de Sanny Builder. Ejecute Sanny con el parametro \debug=X, donde X es una secuencia de caracteres '0' y '1' (sin comillas). Los caracteres significan que opcion de depuracion esta habilitada/deshabilitada.

Ejemplo: \debug=11000

El primer digito 1 habilitara la opcion "code_offsets", el segundo digito habilitara el modo "ignore_unknown". Los digitos del 0 deshabilitaran todas las opciones restantes.
