GTA:SA SFX Directory by pdescobar

The following is a directory of the SFX archives for Grand Theft Auto San Andreas. Each archive is organized by sound bank and includes offsets and all the sampling rates used in the bank for importing as raw data. Soon, the San Andreas Audio Toolkit will be released which will allow easy import and export of these sounds so such information will no longer be needed. The directory is loosely based on the GTA3/VC SFX directories posted to GTAForums by spaceeinstein. Expect wild innacurracies initially as I guessed a lot...

  • GENRL
