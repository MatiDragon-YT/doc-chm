Estableces el tipo de bloqueo de las puertas con: 

020a: set_car $car00 door_status_to 0
ID Bloqueo
1 Desbloqueado
2 Bloqueado
3 Desbloqueado para los NPC
4 Bloqueado
5 Bloqueado (vehiculo policial)
6 Desbloqueado, las puertas permanecen el el vehiculo
7 Bloqueado, las puertas pueden caerse
8 Bloqueado

Cierras todas las puertas, el capo y el cofre del vehiculo con: 

0508: set_car $car00 close_all_doors

Para abrir una Puerta: 

0657: set_car $car01 open_door 0
ID Compuerta
0 Capo (cofre delantero)
1 Baul (cofre trasero)
2 Delantero Izquierda(conductor)
3 Delantero Derecha (pasajero)
4 Trasero Izquierda
5 Trasero Derecha

Estableces la destruccion o la visibilidad de una puerta con: 

0689: set_car $car02 door 2 visible 0

Estableces el reparamiento de una puerta con: 

068A: set_car $car03 repair_door 2

Estableces golpes en una puerta con: 

0730: set_car $car04 damage_door 4

Establece el angulo de una puerta con: 

08A6: set_car $car05 door 2 rotation_to 0.0

Establece el bloqueo de una puerta con: 

0905: set_door $car05 lock 1

Establece el angulo de una puerta y el estado de cierre con: 

095E: set_car $car06 door 1 unlatch 2 angle 0.5
ID Desbloqueo
0 La puerta esta bloqueada en un angulo
1 Se permite que la puerta se abra y se cierre
2 La puerta esta golpeada y bloqueada en un angulo
3 La puerta esta golpeada y se deja abrir y cerrar
4 La puerta se caera del vehiculo y se procesara como quemada(por la explosion del vehiculo)
