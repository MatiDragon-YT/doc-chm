BOOL WINAPI CLEO_RegisterOpcode(WORD opcode, _pOpcodeHandler callback);

Descripcion:
La funcion registra el manejador de codigo de operacion. Si el registro es exitoso, se devolvera VERDADERO. Si a un codigo de operacion con este numero ya se le ha asignado otro controlador, se devolvera FALSE y se mostrara un mensaje sobre posibles problemas en el juego.

Parametro Descripcion
opcode Opcode numero de opcode en el rango de 0x0AF0 a 0x7FFF
callback Puntero de devolucion de llamada a la funcion de controlador de codigo de operacion. Consulte Formato del manejador de codigo de operacion

Ejemplo de uso:

if (!CLEO_RegisterOpcode(0xC00, &opcode_0C00)) {
  return FALSE;
}
