void WINAPI CLEO_WriteStringOpcodeParam(CScriptThread* thread, LPCSTR str);

Descripcion:
La funcion escribe en el operando de una variable de tipo String la cadena que pasa el segundo parametro.

Parametro Descripcion
thread Puntero de estructura de flujo
str Puntero a la cadena que se escribirá en la variable de script.

Ejemplo de uso:

CLEO_WriteStringOpcodeParam(thread, "OK");
