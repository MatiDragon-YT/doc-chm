Una constante es un identificador con un valor predefinido. A diferencia de una variable, el valor de la constante no se puede cambiar en tiempo de ejecución.

Durante la compilación, el nombre de la constante se reemplaza por el valor asociado a él.

Sintaxis

Para definir una nueva constante, usa el constructor CONST..END.

CONST
    <nombre de la constante> = <valor de la constante>
END

Un nombre constante es cualquier identificador permitido (una combinación de letras, números y _).

Hay nombres reservados que no se pueden usar, como Continue, True, And, etc. (consulte compiler.ini).
Los valores constantes pueden ser numéricos(también un identificador de modelo o una etiqueta) y cadenas, e incluso contener una expresión. identificador de modelo, una etiqueta); una cuerda; una expresion (por ejemplo clase o variable); otra constante.

Por ejemplo: 

const
    MoneyRequired = 30
    PlayerMoney = $PLAYER_CHAR.Money
end

if
    PlayerMoney > MoneyRequired
then
    PlayerMoney += -1
end

Durante la compilación, la constante MoneyRequired se sustituye por el número 30 y PlayerMoney por $PLAYER_CHAR.Money.

Los modos de edición pueden cargar constantes desde un archivo externo mediante el parámetro <constants> del archivo games.xml.

Se muestra una lista de las constantes definidas actualmente después de pulsar Ctrl+Space.

Limitaciones

Puede utilizar constantes en cualquier lugar excepto en el caso:

const
    VarName = $Var1
    IndexName = 25
end

VarName[IndexName] = 0

Para compilar dicha expresión, debe escribir el opcode correspondiente, por ejemplo:

0004: VarName[IndexName] = 0

También hay algunas limitaciones con el uso de una expresión como valor constante.
