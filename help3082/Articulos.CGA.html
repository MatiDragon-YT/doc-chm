Tambien puedes crear tus propio grupos de animaciones con animaciones originales del juego o con animaciones creadas por la comunidad, siguiendo estos 3 sencillos pasos.
Recomiendo tener instalado el ModLoader para no remplazar ningun archivo original del juego.

Paso N1
Copiar "animgrp.dat" de la carpeta "data" del juego, a una nueva carpeta ubicada dentro del ModLoader.
Paso N°
Abrir con un editor de texto el "animgrp.dat", en esta nos encontraremos algo parecido a esto:

#
#... etc
#
man, ped, walkcycle, 6
 walk_civi 
 run_civi
 sprint_panic
 idle_stance
 roadcross
 walk_start
end

shuffle, ped, walkcycle, 6
 WALK_shuffle
 RUN_civi
 sprint_panic
 IDLE_STANCE
 roadcross
 walk_start
end
Aqui en donde tenemos los "#" encontraremos la toda la informacion sobre como usar este archivo. Yo se los mostrare traducidos y adaptados para mayor comprencion: 
#
# nombre del archivo:  animgrp.dat
# descripcion: En el archivo describe la asociacion de grupos de animaciones. 
#
# Sintaxis:
#name, filename, animtype, numanims
# animlist....
#end
#
# name:  nombre definido para el grupo de animaciones
# filename: nombre del archivo que contiene las animaciones (de extencion IFP en minusculas)
# animtype: tipo de animaciones (en este caso el tipo es "walkcycle", correspondiendo a las animaciones de apariencia) 
# numanims: cantidad de animaciones en el grupo
# animlist: nombre de la animacion (no inclullen para armas)
#
Tu solo debes crear tu propio grupo al final del texto, luego de la seccion "#Miscellaneous", esto con el unico fin de tener todo mas ordenados y respetar las primeras prioridades.
Un ejemplo que de doy, puede ser con animaciones de parkour que podes hacer que se activen al hacer cierta cosa, como al precionar un nuevo boton asignado para sprintar y hacer trucos.

animgrp.dat 
parkourman, parkour, walkcycle, 6
 parkour_walk_civi
 parkour_run_civi
 sprint_panic
 parkour_idle_stance
 roadcross
 parkour_walk_start
end
cleo.cs 
{$CLEO}
thread 'EJEMPLO'
while true 
   wait 0
   if or
      Game.isPressedVK(0, Key_Vertical)
      Game.isPressed(Key_R)
   then
      wait 0
      Actor.setWalkStyle($PLAYER_ACTOR, "parkourman")
   else
      wait 0
      Actor.setWalkStyle($PLAYER_CHAR, "man")
   end
end

Aunque este es un simple ejemplo que les doy, lo ideal para este mod seria hacer que todas las animaciones cambien mientras se mantiene una tecla precionada [R]
