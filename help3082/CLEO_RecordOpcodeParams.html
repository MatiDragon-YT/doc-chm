void WINAPI CLEO_RecordOpcodeParams(CScriptThread* thread, int count);

Descripcion:
La funcion escribe el numero especificado de valores en la secuencia (hasta 32 piezas a la vez). Los valores se leen del arreglo del opcodeParams. Esta función es conveniente de usar cuando necesita escribir varios parámetros en la secuencia a la vez.

Parametro Descripcion
thread Puntero de estructura de flujo
count El numero de valores escritos en la secuencia.

Ejemplo de uso:

opcodeParams[0].fParam = x; opcodeParams[1].fParam = y; opcodeParams[2].fParam = z;
CLEO_RecordOpcodeParams(thread, 3);
