int WINAPI CLEO_GetOperandType(CScriptThread* thread);

Descripcion:
La funcion obtiene un tipo de operando para un acceso mas seguro a ella.

Parametro Descripcion
thread Puntero de estructura de flujo

Valor de retorno:

Valor Descripcion Ejemplo
globalVar Variable Global $PLAYER_ACTOR
localVar Variable Local 0@
globalArr Arreglo Global $Girl_Weapons(0@, 12i)
localArr Arreglo Local 1@(0@, 12i)
imm8 Entero en el rango de -0x80 a 0x7F 12
imm16 Entero en el rango de -0x8000 a 0x7FFF 260
imm32 Entero en el rango de -0x80000000 a 0x7FFFFFFF 0x969110
imm32f Flotante, Sanny Builder compila como imm32 1.5
vstring Cadena Larga "string"
sstring Cadena Corta 'string'
globarVarVString Variable Globa de Cadena Larga v$12
localVarVString Variable Local de Cadena Larga 0@v
globalVarSString Variable Global de Cadena Corta s$12
localVarSString Variable Local de Cadena Corta 0@s

Ejemplo de uso:

switch (CLEO_GetOperandType(thread))
{
  case globalVar:
  case localVar:
  case globalArr:
  case localArr:
  case imm8:
  case imm16:
  case imm32:
    a = CLEO_GetIntOpcodeParam(thread)
    break;
  default:
    CLEO_SkipOpcodeParams(thread, 1);
}
