Could not decompile script.img

If this error message is given, it means the script.img file is broken or has unknown format. It may also happen if this img file was not compiled along with the main.scm, and the main.scm contains the external scripts data relating to another file. Check if both files are compiled at the same time.

Possible solutions: replace the script.img with the file suitable to the decompiled main.scm.
