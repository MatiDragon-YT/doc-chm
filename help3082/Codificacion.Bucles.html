Un bucle es una declaracion que permite que el codigo se ejecute repetidamente. Sanny Builder admite tres tipos de bucles/ciclo de control: for, while y repeat.

FOR..END

El bucle FOR tiene un número estrictamente determinado de iteraciones (repeticiones).

Sintaxis
FOR <contadora> = <iniciar> TO/DOWNTO <finalizar> STEP <escalar>
END
  • <contadora> - se utilizada una variable como contador para las iteraciones
  • <iniciar> - el valor de la variable del bucle antes de la primera iteración (cualquier valor, incluido el id de modelo)
  • TO ó DOWNTO - incrementar o disminuir el valor de la variable del bucle entre las iteraciones
  • <finalizar> - un valor de la variable del bucle después de la última iteración (cualquier valor, incluido el id de modelo)
  • <escalar> - un valor opcional con el que la variable de bucle aumentará o disminuirá entre iteraciones. Por defecto es igual a 1
var
    $valor : int = 0
    $final : int = 100
end

FOR $contador = 1 to $final step 2
    $valor += $contador
end
Si el contador no se declara con ningún tipo antes del bucle, obtiene el tipo Integer. Si el valor para iniciar, finalizar o escalar, obtienen el mismo tipo que tiene la variable contador. Para utilizar números de punto flotante para los valores inicial y final, declare la variable de bucle con el tipo Float. 
var
    $counter : float
end

for $contador = 1.0 to $final step 2.0
    wait 0
end
Las variables $contador y $final siguen siendo del tipo Float en el bucle.

WHILE..END

Sintaxis
WHILE <condicion del bucle>
    <cuerpo del bucle>
END
  • <condicion del bucle> - un solo opcode condicional
  • <cuerpo del bucle> - comandos para ejecutar en cada iteración; puede omitirse

El bucle WHILE.END funciona mientras la condición del ciclo es verdadera. La condición se evalúa antes de una iteración de bucle. Por tanto, si la condición es falsa, el cuerpo del bucle nunca se ejecuta. 

$var = 10

while $varA > 11
    inc($varA)
end
como la condición del bucle es falsa, inc($varA) nunca se ejecuta.

Las constantes True y False se pueden utilizar como condición de bucle.

while true
    <cuerpo del bucle>
end

Con True el bucle se ejecuta infinitamente, hasta encuentrar el comando Break para detenerse/salir

while false
    <cuerpo del bucle>
end

Con False el compilador ignora este bucle ya que la condición nunca se cumple.

Actualmente, el compilador acepta solo un opcode en la condición del bucle, pero puede verificar más condiciones antes del cuerpo del bucle y usar los comandos Break y Continue. 
while true
    if and
        $varB >= 0
        $varB <= 100
    then
        Break
    end
    
    <cuerpo del bucle>
end
El bucle se ejecuta mientras $varB esté en el rango [0...100]

Break & Continue

Si desea omitir la iteración actual y pasar a la siguiente, utilice el comando Continue.

El comando Break hace que el bucle se detenga de inmediato y proceda a la siguiente orden salida del bucle.

Pueden sustituir un parámetro de código de operación (p.ej., jf Continue) o servir como una declaración independiente. 
while true
    if
        not $actorActual.isDead
    jf Break // salir del bucle

    if
        $actorActual.isDead
    then
        Continue // ir a la siguiente interacción
    end
end

REPEAT..UNTIL

Sintaxis
REPEAT
    <cuerpo del bucle>
UNTIL <condicion del bucle>
  • <cuerpo del bucle> - comandos para ejecutar en cada iteración; puede omitirse.
  • <condicion del bucle> - un solo opcode condicional

El bucle REPEAT..UNTIL se ejecuta hasta que la condición del ciclo devuelve falso. La condición se evalúa después de la iteración, por lo que se garantiza que el ciclo se ejecutará al menos una vez.

Las constantes True y False se pueden utilizar como condición de bucle.

repeat
    wait 1000
until true

Con True el bucle tiene una única iteración. 

repeat
    wait 1000
until false

Con False el bucle se ejecuta infinitamente, hasta encuentrar el comando Break para detenerse/salir

Actualmente, el compilador acepta solo un opcode en la condición del bucle, pero puede verificar más condiciones después del cuerpo del bucle y usar los comandos Break y Continue. 
repeat  
    <cuerpo del bucle>
     
    if and
        $varB >= 0
        $varB <= 100
    then
        Break
    end
until false
El bucle se ejecuta mientras $varB esté en el rango [0...100]
