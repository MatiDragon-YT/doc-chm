Asi como podemos colocar los indicadores para objetos, tambien podemos cambiar sus colores y otras propiedades extras que tienen estas.

Puedes cambiar el color del indicador con: 

0165: set_marker  $marker05 color_to 1

  • 0: Rojo
  • 1: Verde
  • 2: Azul
  • 3: Blanco
  • 4: Amarillo

Puedes cambiar su escala con:

0168: show_on_radar $marker04 size 1

  • < 3 : Reducidas
  • = 3 : Valor Predeterminado
  • > 3 : Ampliadas

Puedes cambiar el color a enemigo o amistoso:

07E0: set_marker $marker04 type_to 1

  • 0 : Rojo Enemigo
  • 1 : Azul Amigable

Y puedes ocultarlo y mostrarlo ;con: 

018B: show_on_radar $marker02 in 1

  • 0 : Esconder
  • 1 : Esconder
  • 2 : Mostrar
  • 3 : Mostrar
  • 4 : Esconder
  • 5 : Esconder
