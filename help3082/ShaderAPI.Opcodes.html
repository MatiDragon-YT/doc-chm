Este es el significado de estas cosas:

%~d% = Cualquier cosa
%~p% = Punto de etiqueta
%~o% = Modelos de objetos de todo tipo
%~g% = Referencia GXT
%~x% = Script Externo
%~s% = Ubicacion de archivo/carpeta
%~h% = Indefinido

=n   = Cantidad de parametros
