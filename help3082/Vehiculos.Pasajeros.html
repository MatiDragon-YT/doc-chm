Entrar

Puedes entrar un actor a un vehiculo al instante o con tiempo al asiento que deses con estos opcodes.

// Al asiento del Conductor 
072A: put_actor $actor01 into_car $car01 driverseat // Instantaneo
05CB: AS_actor $actor10 enter_car $car10 as_driver 3000 ms // Cronometrado
// A un asiento de Pasajero
0430: put_actor $actor02 into_car $car02 passenger_seat 2 // Instantaneo
05CA: AS_actor $actor20 enter_car $car20 passenger_seat 2 time 3000 // Cronometrado
ID Asiento de pasajero
0 Copiloto
1 Pasajero Izquierdo
2 Pasajero Derecho
Sacar

Puedes sacar un actor de un vehiculo de varias formas especificas

05CD: AS_actor $actor01 exit_car $car01
05CF: AS_actor $actor10 exit_car $car10 in_direction 623.6346 854.64 26.352
0633: AS_actor $actor02 exit_car
