DWORD WINAPI CLEO_GetVersion();

Descripcion:
La funcion devuelve el numero de version de CLEO, este valor debe compararse con la constante CLEO_VERSION.

Ejemplo de uso:

if (CLEO_GetVersion() < CLEO_VERSION) {
  MessageBox(HWND_DESKTOP, "Se cargo una version incorrecta de CLEO.", NULL, MB_ICONERROR);
  return FALSE;
}
