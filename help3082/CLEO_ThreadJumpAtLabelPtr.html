void WINAPI CLEO_ThreadJumpAtLabelPtr(CScriptThread* thread, int labelPtr);

Descripcion:
La funcion salta a la etiqueta.

Parametro Descripcion
thread Puntero de estructura de flujo
labelPtr Etiqueta para la transicion.

Ejemplo de uso:

CLEO_ThreadJumpAtLabelPtr(thread, CLEO_GetIntOpcodeParam(thread));
